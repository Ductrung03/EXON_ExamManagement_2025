Portable runtime bundles committed for clone-and-run usage.

Folders:
- EXON.MONITOR
- EXON.GradedEssay
- QuanLyHoiDongThiVer2

How to run:
1. Open the folder for the application you need.
2. Run the matching .exe file in that folder.

Important prerequisite:
- TX Text Control 28 for Windows Forms must be installed for EXON.MONITOR and EXON.GradedEssay.

Why these files are committed:
- Some target machines need a ready-to-run folder instead of rebuilding locally.
- These folders already include EXON.Common.dll, EXON.SubModel.dll, and EXON.SubData.dll.
